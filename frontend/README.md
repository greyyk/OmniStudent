# OmniStudent Frontend

React + Vite frontend for OmniStudent. See the [root README](../README.md) for full setup instructions.

## Quick start
```bash
npm install
npm run dev
```
Runs at `http://localhost:5173`. The backend must also be running on `http://localhost:8000` (the dev server proxies `/api` to it).

## Scripts
- `npm run dev` — start the dev server
- `npm run build` — production build
- `npm run preview` — preview the production build
